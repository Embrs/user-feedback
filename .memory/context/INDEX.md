# 專案知識索引

> `@embrs/user-feedback` v1.3.6 — 基於 Node.js 的 MCP 反饋收集器

## 專案概覽

| 項目 | 值 |
|------|-----|
| 名稱 | `@embrs/user-feedback` |
| 版本 | 1.3.6 |
| 語言 | TypeScript (ESM, ES2022) |
| 執行環境 | Node.js >= 18 |
| 套件管理 | npm |
| 授權 | MIT |

## 核心功能

AI 透過 MCP 協議呈現工作彙報，開啟 Web 介面讓使用者提供文字/圖片反饋，再將結果回傳給 AI。

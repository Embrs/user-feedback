# Change: 開發全新 MCP 反饋收集工具

## Why
基於 `old/` 目錄中的舊版 MCP 工具，需要開發一個功能完整、架構現代化的 MCP 反饋收集工具。現有版本需要優化 UI/UX、強化圖片處理能力、並提供完善的文件與測試支援。

## What Changes
### MCP 核心機制
- **新增** MCP 互動觸發機制（AI 完成回覆/提問/工作總結時自動觸發）
- **新增** 執行流程控制（AI 暫停 → 調用工具 → 收集反饋 → 繼續對話）
- **保持** 現有 MCP 協議 API 相容性

### 使用者介面
- **新增** 左右分割版面設計（左側 AI 工作彙報，右側用戶反饋）
- **新增** 圖片拖放上傳功能，支援順序調整與彈窗檢視
- **新增** 動態罐頭語系統（預設：[無][請繼續][請使用你覺得的最佳解]）
- **新增** 混合回饋支援（圖片 + 文字同時提交）

### 技術規格
- **技術棧** Node.js 18.0.0+、TypeScript、模組化架構
- **新增** 跨平台相容性（Windows、macOS、Linux）
- **新增** Docker 支援（可選）
- **保持** 參考 `old/` 現有實作進行重構

### 文件系統
- **新增** README.md 安裝使用說明
- **新增** API 文件（MCP 介面說明）
- **新增** 開發文件（架構設計與開發指南）

## Impact
- **影響規格**: mcp-core（新增）、feedback-ui（新增）、documentation（新增）
- **影響程式碼**: 全新開發，參考 `old/src/` 結構
- **影響依賴**: Node.js 18+、現有 npm 依賴
- **向後相容**: 保持 MCP 協議介面相容
